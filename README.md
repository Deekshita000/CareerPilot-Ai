# CareerPilot AI

CareerPilot AI is an AI-powered interview preparation and career guidance platform designed to help students prepare for placements efficiently.

## Features
- AI Mock Interviews
- Resume Analysis
- Personalized Feedback
- Role-Based Questions
- Career Guidance
- Modern Dashboard UI

## Tech Stack
- React.js
- Tailwind CSS
- Flask
- SQLite
- AI/NLP
